About Blockchains.io

Blockchains.io is an example web application built using components from the Blockstrap stack. Here you can search blocks, addresses, and transactions for the Bitcoin, Dogecoin & Litecoin blockchains.

For news on updates and improvements follow us on Twitter - @blockchains

About Blockstrap

Blockstrap is a HTML 5 framework and robust API made specifically for developers building on the blockchain. It's core framework is open source and entirely free to download.
